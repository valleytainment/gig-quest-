# `styles/` — Design System CSS

> Layer: ⚪ STYLE | Imported via `src/index.css`

| File | Responsibility |
|------|----------------|
| `tokens.css` | CSS variables — colors, spacing, radii, typography |
| `effects.css` | Gradients, glass panels, shadows (`elite-bg`, `gq-card`) |
| `motion.css` | Animation utilities (`gq-fade-in`, transitions) |
| `forms.css` | Form field styling for intake |

**Rule:** Prefer tokens over hardcoded values in new components.
